# Retail-Inventory-Peformance-Analysis

-- Project overview -- 
This Project focuses on solving inventory and revenue-related business problem. using SQL. The analysis was performed in SQL Server Management Studio (SSMA) using imported retail product data. 

## The project includes:
- Demand analysis
- Discount impact analysis
- Revenue contribution analysis
- category-wise ranking using window functions
- business-focused SQL querying
----------------------------------------------------------------------------------------------------------------
 ## Business Pronblems Sovled:
   ### 1. Demand vs Low Stock Risk
 Identified high-demand products with low stock availability to support inventory restocking decisions
   ### 2. Discount Impact on Customer Engagement
 Analysed how different discount ranges influence customer engagement throughreviews and rratings.
   ### 3. Brand Revenue Contribution
 Calculated revenue contribution by brand to identify high-performing brands contributing significantly to overall business revenue.
